# Dominoes multiplayer online game

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run start:prod
```

## For running online game you need to run Domino-api on your server

[Domino-api](https://github.com/UtkinOleg/domino-api) based on [NestJS](https://github.com/nestjs/nest).
